- [ ] Password manager 🔽 #agent 
  - Answer: Use Bitwarden unless you already rely heavily on an Apple-only workflow. It is cross-platform, has a solid free tier, supports browser/mobile autofill, and keeps export/migration simple. Set up one strong master password, enable 2FA with an authenticator app, store recovery codes somewhere offline, then migrate passwords gradually by starting with email, bank/ID, cloud, GitHub, and other high-value accounts.
