## Plugins

heyo

- **Templater:** Advanced templating and automation using handlebars-like syntax.
  Example: Create a meeting-note template and insert the current date with 
- <% tp.date.now("YYYY-MM-DD") %>.


- **Code Editor Shortcuts:** Adds common code-editor keyboard shortcuts (VSCode/Sublime-style).
  Example: Use `Ctrl-/` to toggle line comments like in VSCode.
- **Various Complements:** IDE-style word autocompletion.
  Example: Start typing a long phrase and accept the autocomplete suggestion to speed writing.
- **Linter:** Formats and styles notes (YAML, headings, lists, spacing, math blocks, etc.).
  Example: Run the linter to fix frontmatter spacing and heading levels before committing.
- **Map View:** Interactive map view for geospatial note visualisations.
  Example: Tag location notes with `#place` and open Map View to see them as pins.
#place 
- **Git:** Integrates Git version control with automatic backups and related features.
  Example: Commit and push vault changes from within Obsidian to keep backups synced.
- **View Count:** Tracks view counts for each file in the vault.
  Example: Sort notes by view count to quickly find your most-visited pages.
- **Calendar:** Calendar view for daily notes.
  Example: Click a date in the calendar to open or create that day's daily note.
- **Auto Template Trigger:** Automatically apply or prompt for templates when creating notes.
  Example: Configure it to auto-apply your meeting template when a new note is created in `Meetings/`.
- **Notebook Navigator:** Two-pane file explorer with tag browsing, previews, and keyboard navigation.
  Example: Open Navigator to filter by `#project` and preview files side-by-side.
- **QuickAdd:** Quickly add pages or content (macros, templates, and quick actions).
  Example: Create a QuickAdd capture that inserts a pre-filled task into `Inbox`.
- **Tasks:** Track and query tasks across the vault (due dates, recurring, filters).
  Example: Use a `Tasks` query block to list all open tasks with `status: TODO` across the vault.
- **Super Simple Time Tracker:** Lightweight time-tracking tools for notes.
  Example: Start a timer on a research note to record time spent on a task.
- **Dataview:** Query and render complex data views from your notes.
  Example: Use a Dataview code block to list notes where `status = "active"`.
- **Copy Inline Code:** Click to copy contents of inline code elements.
  Example: Click an inline code snippet like `npm install` to copy it to clipboard.
- **Doubleshift:** Open command palette by pressing Shift twice and create custom shortcuts.
  Example: Press Shift twice to open the palette and run a custom "Insert meeting template" command.
