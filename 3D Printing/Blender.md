# Set units
1. Go to **Scene Properties > Units**, change Length to **Millimeters**
2. Set **Unit Scale to 0.001**
![[Pasted image 20251220224957.png]]
3. Go to **Viewport Overlays**
4. ![[Pasted image 20260111210223.png]]
5. Set scale to **0.001**
